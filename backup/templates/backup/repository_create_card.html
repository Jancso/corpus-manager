Create Repository
  1. Create a repository on the bot's GitLab account by clicking on New project. Give it a name e.g. 'database-backup'.
  2. Make sure the visibility level of the repository is private!
  3. Tick Initialize repository with a README.
  4. Click Create Project
  5. Copy the repository's URL by clicking on Clone, then Clone with SSH
