The backups will be automatically performed by a bot. Please, follow the instructions to set the bot up:

  1. First create a GitLab account for the bot. For security reasons, do not use your own personal GitLab account!
  2. Click on the avatar (profile pic), click Settings and then SSH keys.
  3. Copy the public key below:
    {{ public_key }}
  4. Paste the public key in GitLab.
